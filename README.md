# Getting Started with BiqQuery and SQL
This is a personal reference for handling large data sets with BigQuery and SQL. Structured Query Language (SQL) is a fundamental tool for working with databases, and mastering SQL is essential for any data professional. In this repository, I curate snippets, scripts, and Jupyter notebooks that allow you to easily harness the power of SQL using BigQuery, a web service that lets you work with massive datasets effortlessly.
